# Logo Files Needed

Please add these logo files to this directory (`client/public/`):

1. **favicon.png** (square logo, recommended 32x32px or 64x64px)
   - Used as the browser tab icon/favicon
   - Should be a square image

2. **logo.png** (landscape logo)
   - Used in the header/navigation
   - Should be a horizontal/landscape oriented image
   - Recommended height: 40px (width will scale proportionally)
   - The logo should display "iCollect" text or branding

After adding these files, they will automatically be used in the application.
